## 📦 Skills 概览

### 1. backup - 配置备份与恢复

**作用**：备份和恢复 OpenClaw 配置及数据

- **核心功能**：使用 git、tar、rsync 等工具进行系统备份和跨设备同步
- **应用场景**：
  - 保护配置、技能、命令和设置
  - 支持灾难恢复
  - 实现多设备间的配置同步

---

### 2. skill-vetter - 技能安全审查

**作用**：安全审查技能包，防止安装恶意代码

- **核心功能**：检查技能代码中的潜在风险
  - 危险操作检测
  - 权限请求分析
  - 数据传输监控
- **应用场景**：在安装第三方技能前进行安全评估，有效防范恶意代码

---

### 3. deepwork-tracker - 专注工作追踪器

**作用**：追踪工作状态，实时了解 AI 代理的工作进展

- **核心功能**：
  - 记录开始/停止时间
  - 统计每日/每周工作时长
  - 生成工作热力图
- **应用场景**：帮助用户追踪专注工作的时间和频率，分析工作效率
- **⚠️ 已修复问题**：解决了不同设备的 sqlite3 路径问题，复制交给 OpenClaw 会自动处理

---

### 4. task-status - 长任务进度反馈

**作用**：为长时间运行的任务提供实时进度反馈

- **核心功能**：
  - 手动发送单次状态更新
  - 启动周期性监控（每 5 秒发送 "Still working..." 消息）
- **应用场景**：让用户实时了解长任务的执行状态，提升使用体验
- **✅ 最新更新**：已升级为通过钉钉发送消息，可在对话中看到实时更新

---

### 5. skill-search - 技能搜索与发现

**作用**：从 ClawdHub 搜索和发现所需技能

- **核心功能**：
  - 搜索可用技能
  - 查看技能趋势
  - 获取技能详情和内容
- **应用场景**：帮助用户发现新技能，无需永久安装即可临时使用技能功能

---

### 6. youtube-clipper - YouTube 视频智能剪辑

**作用**：YouTube 视频的 AI 智能剪辑与处理工具

- **核心功能**：
  - 下载视频和字幕
  - AI 分析生成精细章节（2-5 分钟级别）
  - 自动剪辑视频片段
  - 翻译字幕为中英双语
  - 烧录字幕到视频
  - 生成总结文案

- **工作流程**（6 个阶段）：
  1. 环境检测
  2. 下载视频
  3. AI 分析章节
  4. 用户选择片段
  5. 剪辑处理
  6. 输出最终结果

- **特色亮点**：AI 驱动的精细章节分析，支持 2-5 分钟级别的精准切分

## 🚀 快速安装指南

### 安装步骤

1. **复制技能文件**
   ```bash
   # 将以上 6 个技能文件夹复制到 OpenClaw 技能目录
   cp -r backup skill-vetter deepwork-tracker task-status skill-search youtube-clipper ~/openclaw/skills/
   ```

2. **启动 AI 机器人**
   ```bash
   # 启动 OpenClaw
   pnpm openclaw gateway --verbose
   ```

3. **自动配置**
   - AI 机器人会自动识别新技能
   - 使用 `skill-vetter` 判断安全性
   - 自动补全依赖项

4. **开始使用**
   - 配置完成后即可使用所有新技能

---

## 📝 注意事项

- 首次安装时，建议使用 `skill-vetter` 对所有技能进行安全审查
- 部分技能可能需要额外的系统依赖，请根据提示安装
- 建议定期使用 `backup` 技能备份配置和数据
- 遇到问题可查看各技能文件夹中的 README 文档
