# 获取 B站 Cookie - 图文教程

> 请按顺序查看以下截图

## 第一步：登录 B站

用浏览器打开 https://www.bilibili.com ，使用手机号/邮箱登录你的账号。

## 第二步：打开开发者工具

**Windows/Linux**: 按 `F12` 或 `Ctrl + Shift + I`
**Mac**: 按 `Cmd + Opt + I`

## 第三步：找到 Cookie

1. 点击顶部的 **Network（网络）** 标签
2. 刷新页面（按 F5）
3. 点击任意一个请求（通常是第一个）
4. 找到 **Request Headers（请求头）**
5. 找到 **cookie:** 这一行

## 第四步：复制完整 Cookie

1. 选中 `cookie:` 后面的所有内容
2. 复制（Ctrl+C / Cmd+C）
3. 粘贴到 `cookie.txt` 文件中
4. 覆盖文件中的示例内容

## 注意事项

- 必须登录账号才能获取完整 Cookie
- Cookie 有效期约 1 个月，过期后需要重新获取
- 不要泄露你的 Cookie 给他人
