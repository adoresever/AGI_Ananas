## 📦 Skills 概览

### 1. Bilibili-Study - B站视频学习笔记生成器

**作用**：备份和恢复 OpenClaw 配置及数据

- **核心功能**：将B站视频转化为口语化、可朗读的学习笔记 • 自动下载字幕并生成结构化学习笔记
- **应用场景**：
  - 保护配置、技能、命令和设置
  - 支持灾难恢复
  - 实现多设备间的配置同步

---

### 2. Humanizer - AI写作去AI化

**作用**：识别和移除AI写作痕迹 • 基于维基百科的"AI写作迹象"指南 • 检测24种AI写作模式

- **核心功能**：去除过度强调重要性词汇
  - 修正英中混用术语
  - 简化公式化结构
  - 增加真实感和个性化表达
  - 语言自然化处理
- **应用场景**：去除AI写作痕迹，自然化表达

---

### 3. pdf2video - 专注工作追踪器

**作用**：将 PDF 文档转换为具有流畅动画效果的引人入胜的视频演示文稿

- **核心功能**：
  - 多种场景类型
  - 智能动画
  - 标题系统
  - 底部信息栏
  - 高质量渲染
- **应用场景**：将报告、文档转换为吸引人的视频演示
- 学术论文或商业报告的可视化展示
- 产品手册或说明书视频化
- 教育资料多媒体转换

## 🚀 快速安装指南

### 安装步骤

1. **复制技能文件**
   
   ```bash
   # 将以上 3 个技能文件夹复制到 OpenClaw 技能目录
   cp -r backup bilibili-study humanizer pdf2video ~/openclaw/skills/
   ```
2. **启动 AI 机器人**
   
   ```bash
   # 启动 OpenClaw
   pnpm openclaw gateway --verbose
   ```
3. **自动配置**
   
   - AI 机器人会自动识别新技能
   - 使用 `skill-vetter` 判断安全性
   - 自动补全依赖项
4. **特别注意**
   
   - bilibili-study技能配置较复杂，建议先阅读bilibili-study/README.md和bilibili-study/获取cookie步骤

---

## 📝 注意事项

- 首次安装时，建议使用 `skill-vetter` 对所有技能进行安全审查
- 部分技能可能需要额外的系统依赖，请根据提示安装
- 遇到问题可查看各技能文件夹中的 README 文档
